# Setup Instructions

You need to install NodeJS with NPM, go to [https://nodejs.org](https://nodejs.org/)

1. run `npm install`.
2. run `npm run start`.
3. open http://127.0.0.1:8080

If you need help setting SourceJS help, can can ask for help at the Gitter chat:
